CD와 책의 내용 검색

아래의 박스에 찾고자 하는 키워드를 하나 이상 입력하세요. (영문만 검색 가능)

참고:

  • 검색 키워드를 포함한 책의 페이지 번호도 검색 결과에 포함된다. 책의 페이지 결과는 링크로 연결되지 않는 반면 CD상의 내용은 링크된다.
  • 검색 엔진은 영문에 최적화돼있어 한글 검색은 불가능합니다.
  • 와일드카드 '*'와 '?'를 지원한다.
  • 검색 결과는 순위에 따라 정렬된다. 예를 들어 검색 키워드가 제목에 있는 문서가 본문에만 키워드를 포함하고 있는 문서보다 순위가 높다.
  • PDF 문서를 볼 때 Adobe Reader의 검색 기능을 사용해서 문서 내부 검색을 수행할 수 있다.
