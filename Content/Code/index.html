코드 예제

이 부분에는 새로 보는 프로그래밍 언어의 소스 코드 중 자명하지 않은 것들을 담았다

  • 대부분의 경우 책의 코드가 완전히 실행 가능한 프로그램을 구성하지는 않기 때문에 코드를 확장해야 한다. 일반적으로 현재 다루는 기능을 설명하는 최소한의 프로그램을 생성하는 것이 목표다. 결과 프로그램이 항상 좋은 프로그래밍 스타일이지는 않다.
  • 일부 경우(일반적으로 컴파일러 자체의 동작이 관심사인 경우) 예는 단일 모듈이며 실행 가능한 코드를 생성하지 않는다. 이러한 코드는 “-c” 명령행 스위치를 사용해서 컴파일해야 한다.
  • 소스 파일명과 주석은 책에서 그에 대응되는 예, 그림, 연습문제를 나타낸다. 예, 연습문제, 그림 모두에 나온 코드는 예를 따라서 파일명을 정했다.

코드 예제는 장 별로 나눴다. 예제가 없는 장은 회색으로 나타냈다. 각 장의 내용을 보이거나 감추려면 책 아이콘을 클릭하면 된다.
하위 목차 열기 / 하위 목차 닫기 버튼을 이용해서 전체를 보이거나 가릴 수 있다.
