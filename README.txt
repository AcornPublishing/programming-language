	시스템 요구사항

	  PDF 리더

CD에는 Adobe® Acrobat®이나 Adobe Reader®와 같은 PDF 리더로 열 수 있는 PDF 문서가 
담겨있다. CD에 몇몇 플랫폼을 위한 Adobe Reader의 최신 버전을 담았다.

CD에 담긴 Adobe Reader를 설치하려면 여기를 클릭. 
Adobe Reader에 대해 좀 더 자세한 정보를 얻기 위해 홈페이지를 방문하려면 여기를 클릭 

	   HTML 브라우저

CD의 전반적인 인터페이스와 일부 내용은 HTML와 자바스크립트로 작성했다. CD를 잘 이용하기 
위해 가장 선호하는 HTML 브라우저의 최신 버전을 설치하는 것이 좋다. CD의 내용은 운영체제 
별로 다음과 같은 브라우저를 사용해 검증했다.

윈도우 2000:

	* 인터넷 익스플로러 6.0 
	* 파이어폭스 1.0.6 
	* 모질라 1.7.11 
	* 넷스케이프 7.1, 8.0 
	* 오페라 8.02 

맥 OS X (팬더):

	* 인터넷 익스플로러 5.2 
	* 파이어폭스 1.0.6 
	* 모질라 1.7.11 
	* 사파리 1.3 

만드리바(Mandriva) 리눅스:

	* 파이어폭스 1.0.6 
	* 캉커러(Konqueror) 3.2.3 
	* 모질라 1.7.11 

CD의 내용은 너비가 최소 720픽셀인 브라우저 창에 최적화 되어있다. 
화면 해상도가 1024x768보다 낮을 경우 내용을 제대로 보지 못할 수 있다. 

	   운영체제

이 CD는 HTML 브라우저와 PDF 리더를 포함하는 어떤 운영체제에서도 사용할 수 있다. 
예를 들어 윈도우, 맥 OS, 대부분의 리눅스와 유닉스 시스템이 있다.

	내용의 위치

좌측 탐색 사이드바에는 범주에 따라 CD 내용에 접근할 수 있는 버튼이 있다.

검색 페이지에는 키워드를 포함하는 내용의 위치를 찾기 위한 검색 엔진이 있다. 
검색 페이지로 이동하려면 검색 버튼을 클릭하면 된다.

CD의 내용이 주로 Adobe PDF(Portable Document Format)이기 때문에 PDF 리더의 
검색 기능을 이용해서 각 문서 별로 검색할 수 있다. PDF 파일 내 검색에 대한 정보는 
사용 중인 PDF 리더의 도움말 메뉴에서 얻을 수 있다.

	  페이지, 그림, 연습문제 번호

책과 CD 모두 내용을 쉽게 찾을 수 있는 번호 지정 방식을 사용했다.

	    페이지 번호

책의 페이지 번호는 순차적이다(1, 2, 3 등). 페이지 번호나 페이지 범위(예를 들어 "203쪽", 
"134~145")는 책의 해당 페이지를 의미한다. 

CD의 페이지 번호 역시 순차적이다. 하지만 책의 번호와 구별하기 위해 페이지 번호 뒤에 
"-CD"를 덧붙였다(예를 들어 156-CD). CD의 페이지 번호는 전체 CD에 걸쳐 이어진다.

CD에서 CD 내용에 대한 링크를 표시할 때는 쪽 범위도 함께 나타냈다:

   Section 2.3.4 구문 오류           [1-CD   to 11-CD]
   Section 2.4 이론적 기초          [12-CD to 16-CD]

	    그림과 예

책의 그림과 예의 번호도 순차적이지만 다음과 같이 장 별로 구별했다:

   그림 1.1, 그림 1.2, 그림 1.3 등
   예 1.1, 예 1.2, 예 1.3 등

CD의 번호는 책의 마지막 번호 다음에서 시작한다. 예를 들어 책에 담긴 2장의 마지막 그림이 
그림 2.32라면 CD에 담긴 2장의 첫 번째 그림 번호는 2.33이다. 예의 번호도 마찬가지 방식이다.

	    연습문제와 탐구문제

연습문제와 탐구문제도 그림이나 예처럼 책에 담긴 각 장의 번호 순서를 이어서 CD에 수록했다.

CD에 수록된 연습문제와 탐구문제에 대한 링크를 표시할 때는 쪽 범위를 지정하지 않았다. 한국어판의 
경우에 책에도 심화 연습문제와 탐구문제를 수록했다:

   연습문제 2.24 ~ 2.28 
   탐구문제 2.37 ~ 2.38

	CD 내용을 하드 드라이브에 복사하는 방법

CD 내용에 좀더 빠르게 접근하거나 CD 드라이브를 비우기 위해 CD 내용을 하드 드라이브에 복사할 수 
있다. 복사에는 약 150MB의 용량이 필요하다. 

윈도우 시스템에서는 우선 바탕화면의 내 컴퓨터 아이콘을 더블클릭한다. 그 다음 CD 드라이브(주로 D:) 
아이콘을 우클릭하고 열기 팝업 메뉴를 선택한다. CD의 모든 내용을 선택한 후 명령에서 편집->복사를 
선택한다. 이제 뒤로 버튼을 클릭한 후 CD 내용을 보관하고 싶은 하드 드라이브의 폴더로 이동한다. 
편집->붙여넣기 메뉴를 선택한다. 내용의 복사가 완료되면 하드 드라이브의 index.html을 더블클릭해서 
내용에 접근할 수 있다.

맥에서는 우선 파인더를 사용해서 내용을 복사하고 싶은 하드 드라이브 폴더를 연다. 그 다음 CD를 
나타내는 바탕화면의 PLP2E 아이콘을 더블클릭한다. CD의 모든 내용을 선택한 후 하드 드라이브의 
폴더로 드래그한다. 내용의 복사가 완료되면 하드 드라이브의 index.html을 더블클릭해서 내용에 
접근할 수 있다.

	알려진 문제

2005년 10월까지 보고된 문제

	1. 외부 사이트로의 링크(예를 들어 언어 자료 페이지)를 클릭하는 경우 해당 사이트에서 자바
           스크립트를 사용해서 탐색 사이드바를 없앨 수 있다. 보통 브라우저의 “뒤로” 버튼을 클릭
           해서 CD로 돌아갈 수 있지만 어떤 경우에는 CD로 돌아가지 못할 수 있다. 이러한 동작은 
           윈도우 시스템의 인터넷 익스플로러 6에서 보고됐다. 이 문제를 해결하려면 브라우저를 
           재시작해야 한다.
	2. 탐색 사이드바의 버튼을 클릭할 때 내용이 현재 브라우저 창의 메인 프레임 대신 새로운 
           브라우저 창에 뜰 수 있다. CD 제작 시 이 문제를 알고는 있었지만 이 현상의 발생이 일관
           적이지 않아서 해결하지 못했다. 이 문제는 윈도우 시스템의 인터넷 익스플로러 6에서 보고
           됐으며 해결하려면 브라우저를 재시작해야 한다.
